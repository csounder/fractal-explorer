What is Dr.C - Fractal Explorer?

Dr.C - Fractal Explorer is a professional web-based instrument that transforms L-system fractals into music in real-time. Using Csound WASM 7 for professional audio synthesis, it maps fractal geometry (depth, angle, position) to synthesis parameters, creating organic, evolving soundscapes.

Designed for both algorithmic composition and live performance, it includes 57 factory presets, 8 synthesis engines, master volume control, and comprehensive parameter mapping. All processing runs client-side in your browser — no installation required.

🌀 5 Fractal Types

Algae, Tree, Dragon Curve, Koch Snowflake, and Sierpinski Triangle. Each with unique geometric properties that create distinct musical patterns.

🎹 8 Synthesis Engines

Karplus-Strong Pluck, FM Bell, Additive Bell, Clarinet, Organ, Granular Synthesis, Complex FM (4-operator), and Noise & Filters.

🎛️ Parameter Mapping

Map fractal properties to pitch, duration, timbre, pan, attack, release, reverb room size, and reverb wet mix. Real-time control over all parameters.

🏭 57 Factory Presets

Curated presets: 15 fractal-based, 24 instrument-based, 5 extra noise presets, and 13 fast tempo (⚡) variations. Each showcases unique mappings and sonic characteristics. Auto-play enabled for live performance.

🔊 Csound WASM 7

Professional audio synthesis using Csound 7 in the browser. Includes freeverb reverb, 1st-order ambisonic spatialization, and polyphonic voice management.

🎨 Real-time Visualization

Watch fractals grow in real-time with playback highlighting. Current note cursor and parameter overlay show exactly what's playing.

⚙️ Technical Specifications

  • Audio Engine: Csound WASM 7.0.0-beta26
  • Sample Rate: 44.1kHz, 32 samples/block, stereo output
  • Instruments: 8 synthesis engines (5 melodic + 3 textural)
  • Effects: Freeverb reverb, 1st-order ambisonic spatialization
  • Fractals: L-system generator with 5 built-in curves
  • Mappings: 8 fractal-to-synthesis parameter mappings
  • Master Volume: Real-time control (0-100%)
  • Polyphony: 1-32 voices (user adjustable)
  • Tempo Range: 30-300 BPM with max 2000 events
  • Factory Presets: 57 total (15 fractal + 24 instrument + 5 noise + 13 fast)
  • Preset System: localStorage-based with save/load/delete
  • Browser Support: Modern browsers with Web Audio API
